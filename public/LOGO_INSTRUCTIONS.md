# Instructions pour ajouter le logo

## Ajout du logo

Pour ajouter le logo "Plateforme Citoyenne Hyèroise" :

1. Placez votre fichier logo dans le dossier `public/` :
   - `public/logo.svg` (recommandé, format vectoriel)
   - ou `public/logo.png` (si SVG non disponible)

2. Le Header détectera automatiquement et affichera votre logo.

3. Si le logo n'est pas trouvé, un placeholder "PCH" s'affichera temporairement.

## Nom du logo

Le logo devrait s'afficher avec le texte "Plateforme Citoyenne Hyèroise" en dessous dans le Header.

## Couleurs

Les couleurs du design ont été adaptées pour correspondre au teal-vert de votre logo (#4CA79F). Si vous souhaitez ajuster davantage les couleurs, modifiez `tailwind.config.ts`.
